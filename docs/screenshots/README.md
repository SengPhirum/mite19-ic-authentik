# Screenshot Checklist

Store assignment evidence screenshots or screen-recording exports in this folder.

Recommended filenames:

1. `01-docker-compose-ps.png` - all containers running.
2. `02-initial-setup.png` - initial setup or akadmin bootstrap evidence.
3. `03-users-groups.png` - demo users and groups.
4. `04-admin-totp-enrolled.png` - administrator TOTP enrollment.
5. `05-grafana-provider.png` - OAuth2/OIDC provider settings.
6. `06-grafana-login-flow.png` - redirect from Grafana to authentik.
7. `07-grafana-sso-success.png` - successful SSO as `demo-user`.
8. `08-conditional-access-denied.png` - denied login as `demo-outsider`.
9. `09-events-logs.png` - authentik event/audit log entries.
10. `10-backup-restore.png` - backup file and successful restore output.
